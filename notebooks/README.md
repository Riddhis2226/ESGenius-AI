# 📓 NOTEBOOK DESCRIPTIONS

This section contains all the notebooks required for the project, made in Jupyter Notebook.

## 📘 1. data_connection.ipynb

### Description: Establishes connection with MongoDB, imports ESG datasets, and performs initial data extraction and storage. This notebook handles database setup and prepares raw data for further processing.

## 📘 2. ml_pipeline.ipynb

### Description:Implements the complete machine learning pipeline including data preprocessing, feature engineering, ESG score normalization, and risk classification using Random Forest. The trained model is saved for deployment.

## 📘 3. esg_compliance_ai.ipynb

### Description: Integrates AI-powered ESG analysis with GRI and BRSR framework mapping. Generates intelligent sustainability insights and structured ESG compliance reports using Gemini API.
